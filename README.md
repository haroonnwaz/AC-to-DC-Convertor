<h1>AC to DC Convertor</h1>

<h2>Description</h2>
This project demonstrates my PCB design workflow, from schematic capture to board layout and manufacturing file generation. The board was designed in KiCad with careful attention to component placement, routing, power distribution, and design rule compliance.
<br />


<h2>Features</h2>

- <b>Custom schematic design</b>
- <b>Two-layer PCB layout</b>
- <b>Optimized component placement and routing</b>
- <b>Design Rule Check (DRC) verification</b>


<h2>Environments Used </h2>

- <b>PCB Design</b>
- <b>Electronic Hardware Development</b>
- <b>Schematic Capture</b>
- <b>PCB Layout and Routing</b>
- <b>Design Verification</b>
- <b>Hardware Documentation</b>
